### Лабораторная работа №7. Адаптивная верстка: Flexbox

За время выполнения данной работы были изучены метода адаптивной верстки *Flexbox*, необходимые для того, чтобы контент страницы автоматически подстраивался под размеры окна.

**Содержание работы:**
* `homework` - папка, содержащая `ht.html` и `ht.css`, домашнее задание(создание сайта)
* `img` - папка со скриншотами выполнения шагов лабораторной работы.
* `flexbox.css` и `flexbox.html` - парные файлы, созданные для отработки методов Flexbox. Содержит три примера.
* `index.html` и `styles.css` - парные файлы, созданные для изучения методов работы с Flexbox.
* `README.md` - описание лабораторной работы.
